# Documentation — QapiRag-Jira-Mock

Ce répertoire contient la documentation opératoire et technique pour le mock Jira Cloud v3 basé sur WireMock.

Fichiers importants :
- `quickstart.md` — démarrage rapide et validations.
- `deployment.md` — options de déploiement (local, container, k8s, image immuable).
- `ci.md` — exemple GitHub Actions pour exécuter les tests d'intégration.
- `healthcheck.md` — script de vérification et systemd unit/timer.
- `mappings.md` — conventions des mappings et fixtures.
- `troubleshooting.md` — diagnostics et résolutions courantes.
- `integration-tests.md` — instructions et prompt pour générer les tests MCP.

Consigne : placer ces fichiers dans le repo et les garder à jour avec tout changement de mapping ou d'API attendu par le client.
